# 実装内容

## Event型の定義

`src/types.ts` を新規作成し、03_データ設計 8章のEventに準拠した型を定義する。

```typescript
export type EventStatus = "回答中" | "配車中" | "確定" | "終了";

export interface Event {
  id: string;
  name: string;
  date: string; // "2026-07-12" 形式
  destinationName: string;
  status: EventStatus;
}
```

※ Firestore連携時に必要になる destinationId・latitude・longitude・
createdAt・updatedAt 等のフィールドはTask005以降で拡張する。
本Taskでは画面表示に必要な最小限のフィールドのみ定義する。

## 日付フォーマット

`src/lib/date.ts` を新規作成する。

```typescript
const WEEKDAYS = ["日", "月", "火", "水", "木", "金", "土"];

// "2026-07-12" -> "7/12(土)"
export function formatDateWithWeekday(isoDate: string): string {
  const d = new Date(isoDate + "T00:00:00");
  const month = d.getMonth() + 1;
  const day = d.getDate();
  const weekday = WEEKDAYS[d.getDay()];
  return `${month}/${day}(${weekday})`;
}
```
