## 2. ゴール

未割り当てのグループGを車両Vに配置する際の適合度を評価するため、2地点間のハバーシン距離計算関数と、それを用いた配置適合度スコア計算関数（純粋関数）を実装する。

## 4. 実装範囲

- 緯度・経度を持つ座標情報を表すLocation型（`latitude`, `longitude`）を定義する
- 2地点間の直線距離をハバーシン公式で算出する関数 `getHaversineDistance(point1: Location, point2: Location): number` を実装する（返却単位はkm）
- グループGと車両Vの配置適合度スコアを算出する純粋関数 `calculateScore(group, vehicle): number` を実装する
  - **Distance(G, V)**: 車両のドライバー集合場所（`driverPickupLocation`）とグループの集合場所（`pickupLocation`）の間を `getHaversineDistance` で算出した距離
  - **IsNewStop(G, V)**: グループの集合場所ID（`pickupLocationId`）が車両の経由予定集合場所IDセット（`pickupLocationIds`）に含まれていない場合は1.0、含まれている場合は0.0
  - スコア算出式: `Score = DISTANCE_WEIGHT * Distance(G,V) + STOP_PENALTY_WEIGHT * IsNewStop(G,V)`
- 重み係数の定数 `DISTANCE_WEIGHT = 1.0`、`STOP_PENALTY_WEIGHT = 5.0` をコード上に定義する
- `calculateScore` は引数として渡されたgroup・vehicleオブジェクトの状態を一切変更しない純粋関数として実装する
- `calculateScore` の計算に必要な最小限のプロパティ（`pickupLocation`, `pickupLocationId`, `driverPickupLocation`, `pickupLocationIds`）に限定して、Group型・Vehicle型を定義する

## 5. 実装範囲外

- 緯度経度バリデーション・車両有効定員の初期算出などの前処理（T33で実施）
- 家族グループ形成・ドライバー優先割当バリデーション（T34で実施）
- ソート・貪欲法によるマッチング処理（割り当てフェーズ）（T35で実施）
- `unassignedList` への格納処理・例外系ハンドリング全般（T36で実施）
- `DISTANCE_WEIGHT` / `STOP_PENALTY_WEIGHT` を環境変数・外部設定ファイルから読み込む仕組み（本タスクではコード内定数として定義するのみ。詳細は下記「提案」を参照）
- Group型・Vehicle型が持つスコア計算に不要なプロパティ（`members`, `remainingCapacity`, `driverFamilyId` 等）の定義

---

## 6. 受け入れ条件

- `getHaversineDistance(point1, point2)` が、2地点の緯度・経度からkm単位の直線距離を返す
- `calculateScore(group, vehicle)` が `DISTANCE_WEIGHT * Distance(G,V) + STOP_PENALTY_WEIGHT * IsNewStop(G,V)` の式に基づくスコアを返す
- グループの集合場所IDが車両の経由予定集合場所IDセットに含まれる場合はIsNewStopが0として、含まれない場合は1として計算される
- `calculateScore` を呼び出しても、引数に渡したgroup・vehicleオブジェクトの状態（プロパティ値）が変化しない
- `DISTANCE_WEIGHT`, `STOP_PENALTY_WEIGHT` がコード上の定数として定義され、値の変更が容易な形で分離されている

## 提案（タスク対象外）

設計書には「本値（DISTANCE_WEIGHT・STOP_PENALTY_WEIGHT）はシステム共通の設定（環境変数や外部構成ファイル等）として管理し、コード変更なしで調整可能なパラメータとして扱う」旨の記載がある。本タスクでは1〜3ファイル程度の粒度に収めるため、まずはコード内定数として定義するに留める。環境変数・外部設定ファイル化については、対象範囲・管理方法（.env、Firestore設定コレクション等）を別途検討した上で、別タスクとして切り出すことを推奨する。これは設計変更の提案ではなく、既存記載の実現方法に関する指摘のみである。
