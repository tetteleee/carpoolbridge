# 2. 子供が在籍していないコーチへの対応

## 背景

04_データ設計.md 5章のCoachマスタは、現在「家庭に紐づくコーチ
（保護者自身がコーチを兼ねているケース）」のみを対象としており、
`familyId`は必須項目としている。

検討の過程で「子供がチームに在籍していないコーチ（外部コーチ、
卒団済みだが引き続きコーチをしている親等）」というケースも
想定されたが、現時点のチームには該当者がいないため、MVPでは
対応を見送った。

代わりに、配車画面（Task008a）に「任意の人物を追加」機能
（04_データ設計.md 10章の`type: "other"`）を用意し、
例外的な同乗者はマスタ登録なしの自由入力で都度追加する運用とした。

## 将来対応する場合の方針案

実際にこのパターンのコーチが必要になった場合は、

- `Coach.familyId`を再度任意（nullable）に戻す
- コーチマスタ管理画面（一覧・登録・編集）を新規に作る
  （Task006では家庭登録フォームへの「コーチ名」欄追加のみで
  対応しており、独立したコーチ管理画面は存在しない）
- イベントごとの参加要否をどう回答してもらうか
  （LINE経由の別回答フローが必要になる可能性がある）

を改めて検討する。
